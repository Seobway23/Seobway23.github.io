GestureTrail Privacy Policy

GestureTrail does not collect, store, or transmit any personal data.

All gesture settings are stored locally on your device using Chrome’s storage API and are never sent to any server.

Contact: seobway24@gmail.com

GestureTrail 개인정보처리방침

GestureTrail은 개인정보를 수집·저장·전송하지 않습니다.

모든 제스처 설정은 Chrome 저장소 API를 통해 사용자 기기에만 저장되며, 어떠한 서버로도 전송되지 않습니다.

문의: seobway24@gmail.com

GestureTrail プライバシーポリシー

GestureTrail は、個人データを収集・保存・送信しません。

すべてのジェスチャー設定は、Chrome のストレージ API によりお使いのデバイスにのみ保存され、サーバーへ送信されることはありません。

お問い合わせ: seobway24@gmail.com

GestureTrail 隐私政策

GestureTrail 不会收集、存储或传输任何个人数据。

所有手势设置均通过 Chrome 存储 API 仅保存在您的设备本地，绝不会发送到任何服务器。
